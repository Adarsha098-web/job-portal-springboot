A full-stack Job Portal web application built using Spring Boot, MySQL, and JavaScript. The application allows users to create, search, update, and delete job postings through a simple and responsive interface.

# Features
1. Add new job postings
2. Edit existing jobs
3. Delete jobs
4. Search jobs by title
5. RESTful API architecture
6. MySQL database integration
7. Basic login authentication using Spring Security

# Tech Stack
- Backend: Java, Spring Boot, Spring Data JPA
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Build Tool: Maven

# How to Run
1. Clone the repository:
-  git clone https://github.com/Adarsha098-web/job-portal-springboot.git


2. Open the project in IntelliJ or any IDE.

3. Configure MySQL in `application.properties`.

4. Run the Spring Boot application.

5. Open in browser:
-  http://localhost:8080/
